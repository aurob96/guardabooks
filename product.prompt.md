# Product Prompt

Biblioteca es una PWA personal en espanol para organizar libros fisicos.

Prioridad: uso domestico, claridad, datos confiables y flujos simples.

No construir features nuevas sin preservar catalogo, ubicacion y prestamos.

Toda mejora debe incluir:
- Caso de uso.
- Impacto en datos.
- Impacto movil.
- Prueba manual.
